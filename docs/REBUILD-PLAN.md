# Plan — Split FINALDRAFT.html into a multi-page 11ty site + port the "Inside the Film" & "The Lab" sections

## Context

`FINALDRAFT.html` (the chosen "Option E" design) is a single ~2,500-line scroll page with all CSS/JS inlined. Per the agreed SEO strategy (`SEO-STRATEGY.md`), the site must become a **hub-and-spoke multi-page architecture** so each product, industry, and capability can rank on its own in Google and be cited by AI engines. The current single page can't do that — 12 of 13 products link to `#inquiry`, industries are trapped in a visual accordion, and there are no per-topic URLs.

Separately, the older variant `index.html` contains two strong technical sections missing from FINALDRAFT that add exactly the depth Google/AI reward:

- `#anatomy` — **"Inside the film":** scrollytelling of the 3 laminate layers (print carrier → barrier → seal → bonded film) + laminate-structure chips.
- `#quality` (kicker "The lab") — **"Eight tests. Every batch.":** QC test sheet (GSM, COF, bond, puncture, burst, scrub, heat-seal, tear) + lab-equipment photos + cert chips.

Both `index.html` and `FINALDRAFT.html` share the same design system (identical `:root` CSS variables, DM Serif Display + Space Grotesk), so porting is visually seamless.

## Decisions locked (from user)

1. Add **BOTH** "Inside the Film" and "The Lab" sections — both land on `/quality/` (certifications + 8-test lab sheet + film anatomy = one "how we build it & prove it" page).
2. Static-site generator: **Eleventy (11ty)** — shared nav/footer/head/CSS become reusable partials; the 13 product + 7 industry pages are generated from data via pagination (author template once).
3. URL structure: folders → clean extensionless URLs, e.g. `/products/stand-up-pouches/`.
4. **No `/contact/` page** — inquiry form lives once on the home page, directly above the footer (its current position). Every other page's "Get a quote" CTA links to the home anchor `/#inquiry` (single form, authored once — not duplicated per page). Sitewide footer keeps full NAP; add a Google Map embed + hours into the home inquiry section to recover local-SEO value.
5. **No `/infrastructure/` page** — the 4-machine plant scrolly stays on the home page as-is (not enough standalone content for its own page).

## Target architecture (11ty project)

```
.eleventy.js                 # config: passthrough assets, collections, sitemap, filters
package.json                 # @11ty/eleventy + sitemap plugin
robots.txt                   # allow GPTBot/ClaudeBot/PerplexityBot/Google-Extended; link sitemap
src/
  _data/
    site.json                # single source of truth: NAP, phones, email, hours, certs, MOQ, lead time
    products.json            # 13 formats: slug, name, intro, glance facts, structures, features, markets
    industries.json          # 7 sectors: slug, name, apps, typical build, hero image
  _includes/
    layouts/
      base.njk               # <html> head, nav, footer, JSON-LD, script — every page extends this
      page.njk               # generic content page
      product.njk            # product page template (based on existing product-standup.html)
      industry.njk           # industry page template
    partials/
      head.njk               # per-page title/description/canonical/OG/Twitter + Organization/LocalBusiness JSON-LD
      nav.njk  footer.njk
      inquiry-form.njk       # the ONE inquiry form — included on home only (above footer)
      anatomy.njk            # "Inside the Film" section (ported) → included on /quality/
      lab.njk                # "The Lab / Eight tests" section (ported) → included on /quality/
  assets/
    css/styles.css           # shared CSS extracted from FINALDRAFT (+ anatomy/lab CSS from index.html)
    js/main.js               # reveal IO, formats rail, industries accordion, infra scrolly, anatomy stepIO, ticker, nav progress
    images/                  # ALL images moved off customer-assets.emergentagent.com onto own domain
  index.njk                  # HOME — hero, previews, infra scrolly, why-us, clientele, certs teaser, inquiry (above footer)
  about.njk
  quality.njk                # certifications + {% include lab %} + {% include anatomy %}
  products/products.njk      # products index (lists collection)
  products/product-page.njk  # pagination over _data/products → /products/<slug>/
  industries/industries.njk  # industries index
  industries/industry-page.njk # pagination over _data/industries → /industries/<slug>/
```

Eleventy pagination turns each entry in `products.json` / `industries.json` into its own folder-URL page — template authored once, data supplied 13× / 7×. Sitemap generated by plugin; canonical/OG/schema flow from the data cascade.

## Porting the two sections (from index.html)

- `anatomy.njk` ← `index.html` HTML 2198–2268, CSS 833–1009 + responsive 1738–1769 & 1815–1827, JS stepIO 2729–2742. Lands on `/quality/` (alongside The Lab). Reusable on product pages later.
- `lab.njk` ← `index.html` HTML 2493–2543, CSS `.lab*`/`.test-*` ~1377–1410 (+ its responsive rules), no dedicated JS (uses shared reveal IO). Lands on `/quality/` (replaces FINALDRAFT's thin 3-card certifications block with the 8-test sheet + lab photos + cert chips).
- `/quality/` final composition: certifications intro → The Lab / Eight tests → Inside the Film / anatomy — a substantial "how we build it & how we prove it" page.
- Both use the shared palette/fonts already in `styles.css`, so only the section-specific CSS needs copying. Move their images (`heat.png`, `punture.png`, `cutter.png`, sector images) into `assets/images/`.

## SEO plumbing folded into the build (from SEO-STRATEGY.md)

- Per-page unique `<title>`, meta description, `<link rel=canonical>` — via front-matter in the data cascade.
- JSON-LD: Organization/LocalBusiness (NAP from `site.json`) in `base.njk`; Product per product page (from `products.json`); BreadcrumbList on deep pages; FAQPage where an FAQ block exists.
- Open Graph + Twitter cards + favicon in `head.njk`.
- `sitemap.xml` (plugin) + `robots.txt` allowing AI crawlers.
- Images on own domain, with width/height + descriptive alt (already good in FINALDRAFT).
- Home hero H1 keeps brand voice but the page's first prose binds the primary keyword ("flexible packaging manufacturer … Bharuch, Gujarat").

## Build steps (phased, in order)

1. **Scaffold 11ty:** `package.json`, `.eleventy.js` (passthrough `assets/`, sitemap plugin, products/industries collections). `npm i`.
2. **Extract shared CSS** from FINALDRAFT `<style>` → `assets/css/styles.css`; append anatomy + lab CSS from `index.html`. Extract JS → `assets/js/main.js` (reveal IO, rail drag/arrows, accordion, scrolly, anatomy stepIO, ticker, nav progress).
3. **Build chrome partials** + base layout from FINALDRAFT nav/footer/head.
4. **Home (`index.njk`):** port FINALDRAFT body wholesale (hero, at-a-glance, about teaser, products preview, industries preview, infrastructure scrolly kept, why-us, clientele, certs teaser, inquiry form above footer); convert product & industry cards from `#inquiry`/`#products` anchors into links to the new folder URLs; point all non-home CTAs to `/#inquiry`.
5. **Data + templates:** write `products.json` (seed from FINALDRAFT rail copy + existing `product-standup.html` content model — glance facts, 9 laminate structures, features, markets) and `industries.json`; author `product.njk` / `industry.njk` + index pages; wire pagination.
6. **Static pages:** `about.njk`; `quality.njk` (+ lab include + anatomy include). No infrastructure or contact page.
7. **Assets:** download every `customer-assets.emergentagent.com` image into `assets/images/`; update all srcs. Add favicon.
8. **SEO layer:** `head.njk` meta/OG/canonical, JSON-LD blocks, `robots.txt`, sitemap plugin.
9. **Wire inquiry form** to Formspree (human-quote path — no calculator, per decision); add Google Map embed + hours to the home inquiry section.
10. **Archive old drafts** (`option-a/b/d/e.html`, `hero.html`, `themes.html`, `sectors-*.html`, root `FINALDRAFT.html`/`index.html`) into `/archive/` once content is harvested — keep the repo clean and avoid duplicate-content indexing.
