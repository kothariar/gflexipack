# G Flexipack — website

Marketing site for **G Flexipack**, a flexible-packaging manufacturer in Bharuch, Gujarat (est. 2016, part of the Gangamani group). Custom-printed pouches and film rolls — 9-colour rotogravure, ISO 9001:2015, 300+ MT/month capacity.

Static site built with [Eleventy (11ty)](https://www.11ty.dev/) and deployed to GitHub Pages via GitHub Actions.

## Quick start

```bash
npm install         # install Eleventy (only dependency)
npm start           # dev server with live reload (eleventy --serve)
npm run build       # production build → _site/
```

`npm start` and `npm run serve` are identical. The dev server prints a local URL (default `http://localhost:8080/gflexipack/` — note the path prefix, see below).

**To change site content, edit `src/`** — most copy and data lives in `src/_data/*.json`. Product and industry pages are generated by pagination, so adding an entry to `products.json` / `industries.json` creates a new page automatically.

## Build & config notes

Defined in `.eleventy.js`:

- **Input `src/` → output `_site/`** (`_site/` is gitignored).
- **`pathPrefix: "/gflexipack/"`** — the site is served from a subpath on GitHub Pages, so root-relative URLs are rewritten at build time. If a custom root domain is added, change this to `"/"`.
- Passthrough copy: `src/assets` → `/assets`, plus `robots.txt` and `favicon.svg`.
- `sitemap` collection includes every page except those marked `noindex` or `eleventyExcludeFromCollections`.
- Filters: `inlineFile` (embeds CSS/JS into the one-page builds), `year`, `absUrl`.

## Deployment

Every push to `main` triggers `.github/workflows/deploy.yml`: it runs `npm ci` → `npm run build` and publishes `_site/` to GitHub Pages. No manual deploy step.

## Content is the source of truth

Company facts (capacity, MOQ, certifications, address, phone numbers, etc.) live in **`src/_data/site.json`** — edit there, not in the templates. Current key facts: est. 2016 · 60,000+ sq ft plant · 300+ MT/month · MOQ 500kg · 9-colour rotogravure · ISO 9001:2015 · 14 product formats · 7 industries.
